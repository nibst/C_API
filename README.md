## Description
Making a C REST API, using basics like sockets, phtreads and such. For http headers parsing Im using picohttpparser. For JSON parsing Im using yyjson.


## Database intialization
For initializing db we will use `sqlite3 <db_name.db> < init.sql`
On my initial tests `<db_name.db>` will be jokes.db

## Running 
To compile and run it just type: `make ; ./bin/myserver`
